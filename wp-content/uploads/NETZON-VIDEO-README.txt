This folder should contain the background video used by the homepage.

Requested file: netzon-video.mp4
Local path expected by the site: /wp-content/uploads/netzon-video.mp4

I could not create a binary .mp4 file here via the editor tools. Please copy your real MP4 video file into this location on your machine.

Example (Windows cmd.exe) to copy a file into place:

    copy "C:\path\to\your\netzon-video.mp4" "D:\Websites\@Company\intrusionx\wp-content\uploads\netzon-video.mp4"

Or using PowerShell:

    Copy-Item -Path "C:\path\to\your\netzon-video.mp4" -Destination "D:\Websites\@Company\intrusionx\wp-content\uploads\netzon-video.mp4"

Once the real MP4 is in place, the homepage background video will load from /wp-content/uploads/netzon-video.mp4.
